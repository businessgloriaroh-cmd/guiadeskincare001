## 🖼️ Sobre as imagens

As imagens dos produtos ficam na pasta **`paginas do dicionario/imagens/`**.

Enquanto uma imagem não está disponível, usamos um placeholder como:
```
https://placehold.co/600x400/EEE/31343C
```
Você pode customizar as cores para deixar óbvio que a imagem precisa ser trocada.

---

## ✍️ Como adicionar uma nova aula

1. **Crie um novo arquivo HTML** na pasta `paginas do dicionario/` — nomeie como `002.html`, `003.html`, etc.
2. **Use o mesmo estilo** do `001.html` como base (copie e edite o conteúdo).
3. **Adicione as imagens** na pasta `paginas do dicionario/imagens/`.
4. **Atualize o `index.html`**: substitua o card "Em breve" pelo link da nova aula, e adicione um novo card "Em breve" para a próxima.
